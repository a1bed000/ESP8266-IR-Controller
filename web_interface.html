📺 ESP8266 红外遥控器

智能遥控 · 定时任务 · WiFi 控制

🖥️ 系统状态

正在连接到 ESP8266...
请确保设备已开启并连接到同一网络

⚙️ 服务器设置

输入 ESP8266 的 IP 地址（在串口监视器中查看）

🎓 红外学习

输入命令名称，点击学习，然后在 30 秒内按遥控器按键

建议学习的命令: power, vol+, vol-, source, up, down, left, right, ok, menu, back

🎮 手动控制

电源 音量+ 音量- 信号源 ↑ ↓ OK 返回

▶️ 自动播放

立即执行自动播放序列（开机 → 选择信号源 → 播放）

执行序列: 电源 → 等待3秒 → 信号源 → OK → 下 → 下 → OK

⏰ 定时任务

🌅 每天早上 08:00 自动开机并播放视频
🌙 每天晚上 19:00 自动关机

定时任务依赖 NTP 网络时间，请确保设备能访问互联网
